# Springboot-REST-API
[springboot 기반 REST API 개발](https://www.inflearn.com/course/spring_rest-api/) 강의를 보며 공부하는 repo

이 강좌에서는 다음의 다양한 스프링 기술을 사용하여 REST API를 개발합니다.

* 스프링 프레임워크
* 스프링 부트
* 스프링 데이터 JPA
* 스프링 HATEOAS
* 스프링 REST Docs
* 스프링 시큐리티 OAuth2

테스트 주도 개발(TDD)로 진행합니다.
